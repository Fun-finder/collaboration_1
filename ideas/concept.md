# 백 기능
## 1. 로그인
- 세션 값 저장 
- expired_dt
- 비밀번호 난독화
- kakao, google oauth
- 카카오 연동은 회원가입 이후에 가능하게 
    - 아니면 회원정보를 카카오에 요청해야하는데 조금 복잡해질 듯
- 아이디 기억
## 2. 로그아웃
- 세션이 없을 때
- 로그아웃을 눌렀을 때
## 3. 첫 화면 조회 
- 이미지 썸네일들 출력
- 최대 16개
- 무한 스크롤 + 8씩
- 날짜, 타이틀/줄글 (글자제한), 키워드 (해쉬테그)
## 4. 글 1개 자세히보기
- 원본 이미지 출력, 타이틀/줄글 (제한 없음)
## 5. 글 입력
- 이미지 (단일/ 여러개/ 개수 제한/ 이미지 사이즈 제한)
- 타이틀/ 줄글 - 위협문자 제거
## 6. 중점사항
- secure coding
    - xss 방지
    - sql injection 방지
- 파일 저장 경로 
- 파일 저장 권한 (서버)
- 객체 지향
- 테스트 코드
